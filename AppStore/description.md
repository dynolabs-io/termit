## Long description (4000 chars max)

Termit is a peer-to-peer SSH + Mosh terminal for iOS, built for developers and operators who manage servers from their phone.

What makes Termit different:

**Mosh works on every host, not just the ones with mosh-server already installed.** When you connect to a fresh server that doesn't have it, Termit offers three choices — use SSH for the session, install via the system package manager (sudo), or drop a portable static `mosh-server` binary into your home directory (no sudo needed). This last option works on hardened/shared hosting where you can't install packages — something no other iOS terminal does.

**Your SSH keys live in the Secure Enclave.** Generated on-device, never exfiltrable. Every session requires Face ID or Touch ID. We can't see your keys; iCloud can't see your keys; Apple can't see your keys.

**No mandatory account.** Termit works offline, no signup, no login wall. Optional encrypted iCloud sync if you want hosts/snippets across devices — the encryption key never leaves your device.

**True iOS-native.** Live Activities on Dynamic Island show your active session at a glance. Lock Screen + StandBy widgets one-tap reconnect. Apple Watch app for quick reconnects. Shortcuts/Siri intents ("Hey Siri, connect to prod-edge-1"). Stage Manager + Magic Keyboard chord parity with desktop terminals. 60fps Metal rendering with JetBrains Mono.

**Real terminal features.** SFTP browser with preview/edit. Port forwarding (local/remote/SOCKS). Snippets / saved commands with tag-scoped execution. True-color, OSC52 clipboard, all the things.

**Mosh that just works.** UDP roaming survives the train tunnel, the cell-tower handoff, the wifi-to-5G switch. Predictive local echo for snappy typing across slow links.

**Open source under MIT.** Build it yourself, audit the cryptography, fork it. https://github.com/dynolabs-io/termit

Pricing: free during beta. After 1.0, a one-time pay or affordable subscription — we'll never gate basic SSH/Mosh behind a paywall, and there will always be a free tier.
